HTML Web Design

Click here for the syllabus

This course will teach you the basics of web design using HTML and CSS (Javascript is not discussed). Topics include basic elements, responsive design, and web design best practices. By the end of the course, you will have built your own personal website. Overall, the HTML Web Design course is a great introduction into the basics of both frontend and backend website development.

  • Reverse classroom model
  • $315 CAD
  • 12 weeks
  • A computer with administrative access is required
